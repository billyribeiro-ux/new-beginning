CSS Architecture & Visual Engineering

Volumes 1–4 taught the language and structure of a SvelteKit + Svelte 5 app. This volume is the missing CSS layer — written from the perspective of having actually done the work, the failures included. Every chapter anchors to a specific commit in the project's git history. Every measurement comes from a real pnpm build of the repo, not generic textbook CSS.

The differentiator: this volume narrates the false starts. A cascade-ordering regression that broke production intermittently. Three failed attempts at single-source breakpoints before the working pipeline. A chart-label fix that solved a problem the user didn't have. An icon-library tree-shake that dropped client JS by 77% — once we figured out the wildcard-extension trap that wasn't in any doc. L7 isn't about not making mistakes; it's about catching them quickly, fixing them cleanly, and not pretending they didn't happen.

Fourteen chapters. Every src/lib/styles/*.css file. Every commit since affc7b5. Every measurement re-verified against the current build at write-time. If something in this volume disagrees with the code, the volume is wrong — open a PR.
